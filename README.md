

## Description
Implementation of Dijkstra's algorithm to find the shortest paths from a starting vertex to all other vertices in a weighted graph.

## Requirements Completed
-  Edge class with weight field
-  Weighted graph using adjacency list
-  `void dijkstra(int start)` method
-  Clear output of shortest distances to all vertices
-  No priority queue (simple loops only)
- Arrays for distances and visited nodes



### Compile:
```bash
g++ dijkstra.cpp -o dijkstra
